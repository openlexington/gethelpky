MEDICATION ASSISTED TREATMENTS

Medication Assisted Treatments (MAT)

MAT combines behavioural therapy and medications to treat substance use disorders. These medications may include: buprenorphine (ex: subutex, suboxone, zubsolv), methadone, and naltrexone (Vivitrol). These national locators will help you find providers near you:

  • http://naabt.org (matches prospective patients to providers with availability)
  • http://www.samhsa.gov
  • http://kbml.ky.gov

About Get Help Lex

Get Help Lex is a resource for people seeking facilities and services for substance use disorder (substance abuse/addiction) in and around Lexington, Kentucky. Get Help Lex is an open-source project. GHL is an adaptation of Code for Boston's Finda project. The source code for both Finda and GHL are publically available on GitHub.
Please send feedback, ideas, and bug reports here.

Application creation and support from:

LFUCG Substance Abuse and Violence Intervention Program

The Substance Abuse and Violence Intervention (SAVI) Program works with community stakeholders and advocates in Lexington to increase programming and opportunities that reduce drug related violence by teens and young adults; decrease the stigma that inhibits recovery; advocate for and increase access to effective substance abuse treatment programming.

OpenLexington is a volunteer group of civic-minded software developers, designers, urban planners, citizen activists, data geeks and more. Part of the Code for America Brigade program, we work with LFUCG, Fayette County, and local communities to build open-source civic apps, open public data sources, and generally help to leverage technology for use in the public sphere.

LexLadiesCode

LexLadiesCode is a women's coding club based in Lexington, Kentucky gathered to increase the visibility of and possibilities for women in technology.

Code for Boston

Code for Boston aims to provide a 'center of gravity' for grassroots, community-driven civic innovation in the Greater Boston area. Our work includes building apps, community organizing, policy initiatives, qualitative research, and more, and we hope that volunteers of all types will join our community.
